## Overview

We initiate our project by studying fundamental Optimization tools, including First and Second Variation methods for single and multi-variable scenarios. We progress to exploring Optimal Control Theory, focusing on Controllability, the Bang-bang Principle, Linear TimeOptimal Control, the Pontryagin Maximum Principle, and Dynamic Programming. We adapt real-life examples and formulate numerical solutions and visualization to them. Our final investigation extend from Optimal Control Theory to its utilization in diverse fields such as Game Theory, Stochastic Calculus, and Partial Differential Equations. 

## Theoretical Part - Basic Problem

In this section, we explore the basic problem of controlled dynamics. We start with an ordinary differential equation that describes the dynamical evolution of a system. The unknown function \( \mathbf{x} : [0, \infty) \to \mathbb{R}^n \) represents the state of the system, governed by:

\[
\dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t)), \quad t > 0
\]
\[
\mathbf{x}(0) = \mathbf{x}^0
\]

Here, \( \mathbf{f} : \mathbb{R}^n \to \mathbb{R}^n \) is a given function. We can generalize this problem by introducing control parameters \( a \in A \subset \mathbb{R}^m \), resulting in the system:

\[
\dot{\mathbf{x}}(t) = \mathbf{f}(\mathbf{x}(t), a), \quad t > 0
\]
\[
\mathbf{x}(0) = \mathbf{x}^0
\]

This allows us to study how the system evolves when the control parameter is set to a specific value \( a \).
